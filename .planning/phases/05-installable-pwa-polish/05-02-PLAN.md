<task id="2">
<type>execute</type>
<files>
  - "dashboard"
</files>
<action>

Create a manual test checklist for verifying PWA installability:

1. **Chrome on Android:**
   - Open dashboard URL
   - Look for install banner or "Add to Home Screen" in menu
   - Install and verify app opens standalone

2. **Safari on iOS:**
   - Open dashboard URL
   - Tap Share > Add to Home Screen
   - Verify icon appears and app opens standalone

3. **Desktop Chrome:**
   - Open dashboard URL
   - Look for install icon in address bar
   - Install and verify opens in own window

4. **LAN Testing:**
   - Access from phone on same network using local IP
   - Verify PWA install works over LAN

</action>
<acceptance_criteria>

- Test checklist created
- All install paths documented

</acceptance_criteria>
</task>
