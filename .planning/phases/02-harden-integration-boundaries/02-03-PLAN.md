## Document architecture boundaries

### Create or update architecture docs

The three-way ownership is:
- **Desktop shell** (`src/bun/index.ts`): Stack orchestration, service startup
- **Node bridge** (`bluetti-mqtt-node`): BLE polling, MQTT publishing
- **Python API** (`api/main.py`): MQTT subscribe, REST/WebSocket

### Document in README or ARCHITECTURE.md

Add a clear architecture section:

```markdown
## Architecture

### Component Ownership

| Component | Responsibility | Entry |
|-----------|----------------|-------|
| Desktop shell | Stack orchestration | `npm run desktop:dev` |
| Node bridge | BLE device polling | `bluetti-mqtt-node` CLI |
| Python API | Data service | `api/main.py` |
```
