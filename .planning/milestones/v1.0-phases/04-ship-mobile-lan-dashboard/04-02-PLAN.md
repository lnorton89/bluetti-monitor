## Task 1: Test Overview Page on Phone

On a real phone connected to the same network:
1. Open dashboard URL (e.g., http://192.168.x.x:PORT)
2. Verify hero section shows battery percentage
3. Verify power flow panel shows input/output values
4. Verify device status section is readable
5. Test sidebar navigation opens/closes correctly
6. Test theme toggle works

Expected: Overview page is fully functional on phone.

## Task 2: Test Charts Page on Phone

1. Navigate to Charts page
2. Verify chart renders within viewport
3. Verify chart interactions work (if any)
4. Scroll vertically to see all chart options

Expected: Charts page is usable on phone.

## Task 3: Test Solar Page on Phone

1. Navigate to Solar page
2. Verify both PV inputs display correctly
3. Verify solar generation stats are readable
4. Verify battery charge progress shows

Expected: Solar page stacks vertically and is readable.

## Task 4: Test Raw Data Page on Phone

1. Navigate to Raw Data page
2. Verify table is scrollable horizontally
3. Verify field labels and values are readable
4. Verify search/filter works if present

Expected: Raw data accessible with touch scroll.

## Task 5: Document Test Results

Create a test checklist documenting:
- Device tested (phone model, browser)
- Screen sizes verified
- Any issues found
- Screenshots if helpful
